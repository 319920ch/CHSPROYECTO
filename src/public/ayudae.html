Esta es la pantalla de bienvenida para el usuario con el rol de Encargado

Está compuesto por un mensaje de bienvenida con el nombre del usuario registrado, el logo de la aplicación y un menú de hamburguesa desplegable cada vez que se hace clic sobre las tres rayas horizontales ubicadas en la parte superior derecha.

Para las tareas que realiza el Encargado se muestran dos botones que dirigirán a las páginas correspondientes para la realización de sus actividades.


Esta es la página que se muestra cuando se hace clic en el primer botón de Asignar Empleados

Se trata de una página que realiza el proceso de asignación de empleados para participar en el área correspondiente de un determinado proyecto.


Esta página se muestra al hacer clic sobre el botón Asignar Empleado. Si no se han asignado todavía, se muestra en el campo de Número de empleados recomendados la leyenda Recomendación no encontrada. Para obtener una recomendación se debe hacer clic en el botón Enviar Sugerencia


Una vez que se obtiene la recomendación, se podrá ver en los datos el número recomendado.


Esta página se muestra cuando se hace clic en el botón Obtener Empleados. Es un listado de los empleados ordenado de forma descendente según su desempeño


Cuando se hace clic sobre el botón Asignar, se asigna el empleado al área de trabajo para el contrato especificado .


Cuando se hace clic sobre el botón Guardar asignaciones, se muestra una alerta en caso de éxito.


Al regresar a la página con los datos de recomendación se habrá actualizado el número de empleados actualizados.


Esta página se muestra al hacer clic sobre el segundo botón Estado de Área. Muestra la información de todos los contratos registrados con su estado actual. Su propósito es modificar el estado del área correspondiente.


Esta página aparece al hacer clic sobre el botón Modificar estado. Muestra la información de número de contrato, número de proyecto y estado actual, además permite elegir un nuevo estado y agregar el progreso correspondiente.


Se puede solamente modificar el presupuesto y esto se verá reflejado cada vez que se cargue la página.


Cuando se modifica el estado y se da clic en el botón Guardar, se obtiene un mensaje de confirmación.


En la página de Estado de Área, se visuliza una notificación cuando hay un estado en proceso.


En la página de Estado de Área, se visuliza una notificación cuando hay un estado atrasado. Y se pone un botón para asignar más empleados de modo que se pueda avanzar


Cuando se actualiza un estado a finalizado, automáticamente se actualiza el estado a en proceso para la siguiente área consecutiva.
