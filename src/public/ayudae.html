Esta es la pantalla de bienvenida para el usuario con el rol de Encargado

Está compuesto por un mensaje de bienvenida con el nombre del usuario registrado, el logo de la aplicación y un menú de hamburguesa desplegable cada vez que se hace clic sobre las tres rayas horizontales ubicadas en la parte superior derecha.

Para las tareas que realiza el Encargado se muestran dos botones que dirigirán a las páginas correspondientes para la realización de sus actividades.


Esta es la página que se muestra cuando se hace clic en el primer botón de Asignar Empleados

Se trata de una página que realiza el proceso de asignación de empleados para participar en el área correspondiente de un determinado proyecto.


Se deben llenar todos los campos correspondientes para un registro exitoso de los datos del contrato. Al hacer clic en el botón Registrar Contrato se confirma la creación en un alert en la parte superior de la página.


Una vez que se registra el contrato, automáticamente se muestra una página con un formulario para el registro del proyecto correspondiente. Se deben llenar todos los campos correspondientes para un registro exitoso del proyecto. Al hacer clic en el botón Registrar Proyecto se confirma la creación en un alert en la parte superior de la página.


Una vez que se registra el proyecto, automáticamente se muestra una página con un formulario para asignar el presupuesto por área de proceso de la organización. Para guardar los datos se debe hacer clic en el botón Asignar presupuesto por área.


Esta ventana modal muestra todos los datos del registro de contrato, proyecto y presupuesto de áreas.


Esta página se muestra al hacer clic en el botón Modificar presupuesto de áreas, se trata de una tabla con la información de cada contrato y un botón correspondiente para la modificación de presupuestos.


Cuando se hace clic sobre el botón Modificar Presupuesto se muestra la información de cada área del contrato y proyecto. Cuando se ingresa un nuevo valor y se hace clic en el botón Modificar se actualiza automáticamente en la tabla con el valor ingresado.


Esta página se muestra al hacer clic sobre el tercer botón Estado de Contratos. Muestra la información de todos los contratos registrados con su estado actual. Su propósito es solo informativo.


Esta página aparece al hacer clic sobre el cuarto botón Informes. Muestra la información de todos los contratos registrados con un botón para la generación de informes.


Se debe seleccionar el contrato correspondiente del cual se desea generar el informe. Una vez seleccionado el contrato, se debe hacer clic en el botón Generar PDF. Esto mostrará la información y podrá ser descargada en formato PDF.


Así se visualiza el informe generado.
