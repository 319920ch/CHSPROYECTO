Esta es la pantalla de bienvenida para el usuario con el rol de Administrador

Está compuesto por un mensaje de bienvenida con el nombre del usuario registrado, el logo de la aplicación y un menú de hamburguesa desplegable cada vez que se hace clic sobre las tres rayas horizontales ubicadas en la parte superior derecha.

Para las tareas que realiza el Administrador se muestran tres botones que dirigirán a las páginas correspondientes para la realización de sus actividades.


Esta es la página que se muestra cuando se hace clic en el primer botón de Gestión de usuarios

Se trata de una página que muestra la información de los usuarios registrados, permitiendo acciones como crear nuevos usuarios. De la información disponible se presenta la posibilidad de que se elimine el usuario o se edite la información desplegada.


Esta ventana modal aparece al hacer clic sobre el botón Crear Nuevo Usuario

Muestra un formulario para completar con la información del nuevo usuario.

Se deben llenar todos los campos correspondientes para un registro exitoso del nuevo usuario.


Esta ventana modal aparece al hacer clic sobre el botón Eliminar

Se obtiene el nombre del usuario que será eliminado.


Esta ventana modal aparece al hacer clic sobre el botón Editar

Se obtiene un formulario con la información del usuario que se puede editar.


Esta es la página que se muestra cuando se hace clic en el segundo botón de Gestión de empleados

Se trata de una página que muestra la información de los empleados registrados, permitiendo acciones como crear nuevos empleados. De la información disponible se presenta la posibilidad de que se elimine el usuario o se edite la información desplegada.


Esta ventana modal aparece al hacer clic sobre el botón Crear Nuevo Empleado

Muestra un formulario para completar con la información del nuevo empleado.


Esta ventana modal aparece al hacer clic sobre el botón Eliminar para confirmar la eliminación del empleado seleccionado


Esta ventana modal aparece al hacer clic sobre el botón Editar

Se obtiene un formulario con la información del empleado que se puede editar.


Esta página aparece al hacer clic sobre el botón Desempeño

Se obtiene información de las evaluaciones de desempeño del empleado en las diferentes áreas de producción de la organización


Esta ventana modal aparece al hacer clic sobre el botón Agregar Evaluación de Desempeño

Muestra un formulario para completar con la información de la nueva evaluación.


Esta ventana modal aparece al hacer clic sobre el botón Editar

Se obtiene un formulario con la información de evaluación de desempeño que se puede editar.


Esta alert aparece al hacer clic sobre el botón Eliminar para confirmar la eliminaión de evaluación de desempeño elegida


Esta es la página que se muestra cuando se hace clic en el tercer botón de Gestión de productos

Se trata de una página que muestra la información de los productos registrados, permitiendo acciones como crear nuevos productos. De la información disponible se presenta la posibilidad de que se elimine el producto o se edite la información desplegada.


Esta ventana modal aparece al hacer clic sobre el botón Crear Nuevo Producto

Muestra un formulario para completar con la información del nuevo producto.


Esta ventana modal aparece al hacer clic sobre el botón Editar

Se obtiene un formulario con la información del producto que se puede editar.


Esta ventana modal aparece al hacer clic sobre el botón Eliminar

Se obtiene la información del producto que se puede eliminará.
